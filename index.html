تطبيق الخدمات المتعددة

منصة شاملة لربط مقدمي الخدمات بالعملاء


Moones Mezher

نظرة عامة على المشروع

تطبيق جوال متكامل يعمل على أنظمة Android و iOS، يهدف إلى ربط أصحاب الخدمات (مطاعم، توصيل، هدايا، وغيرها) بالعملاء من خلال منصة احترافية. يوفر التطبيق تجربة مستخدم سلسة لتصفح الخدمات، طلب المنتجات، ومتابعة الطلبات بكل سهولة.

تم تصميم التطبيق ليكون قابلاً للتوسع والتطوير، مع إمكانية إضافة خدمات جديدة ومزايا متقدمة في المستقبل.

المزايا الرئيسية للتطبيق

🎯 للعملاء

  • تسجيل حساب شخصي وتسجيل دخول آمن
  • تصفح الخدمات المتوافرة (مطاعم، عقارات، توصيل، هدايا، وغيرها)
  • عرض الشركات والمحلات المسجلة في كل خدمة
  • تصفح المنتجات والعقارات مع الأسعار والشرح التفصيلي
  • معرفة توفر خدمة التوصيل وتكلفتها
  • شاشة رئيسية تعرض المنتجات الأكثر طلباً
  • محرك بحث قوي للعثور على أي منتج أو خدمة
  • معرفة أوقات العمل وحالة المحل (مفتوح/مغلق)
  • معرفة وقت التوصيل المتوقع
  • قائمة مفضلة لحفظ المنتجات والشركات المفضلة
  • عرض السعر الإجمالي وطرق الدفع المتاحة
  • مشاركة الموقع الجغرافي الحي مع الآخرين عبر رابط فريد (للتوصيل)
  • طلب خدمات التوصيل (تكسي، توصيل طلبات) مع تتبع الموقع

👨‍💼 لأصحاب الخدمات

  • لوحة تحكم شاملة لإدارة الخدمة
  • إضافة وإدارة مدخلي البيانات للرد على الطلبات
  • إدارة المنتجات والأسعار بسهولة
  • إنشاء عروض وهدايا خاصة
  • تحديد أوقات العمل وخيارات التوصيل
  • متابعة الطلبات والتفاعل معها
  • استقبال إشعارات الطلبات على الواتساب (ميزة اختيارية)
  • إحصائيات مفصلة عن المبيعات والطلبات

التقنيات المستخدمة في المشروع

تطبيق الجوال

  • React Native - لتطوير تطبيق موحد لـ Android و iOS
  • Redux - لإدارة حالة التطبيق
  • React Native Maps - للخرائط والتتبع الجغرافي
  • Geolocation API - لتحديد ومشاركة المواقع

الواجهة الخلفية

  • Node.js مع Express.js - لبناء API سريع وآمن
  • MongoDB - لقاعدة البيانات الرئيسية
  • Socket.io - للإشعارات الفورية والتتبع الحي
  • WhatsApp Business API - لإرسال إشعارات الطلبات

خدمات خارجية

  • Cloudinary - لتخزين الصور والملفات
  • OneSignal - لإشعارات Push
  • Twilio / WhatsApp API - لإشعارات الواتساب

التصميم والتجربة البصرية

سيتم تصميم التطبيق بأسلوب عصري وجذاب يعكس الاحترافية، مع مراعاة:

  • ألوان جذابة ومريحة للعين تناسب طبيعة التطبيق
  • تجربة مستخدم سهلة ومبسطة لجميع الفئات العمرية
  • تصميم متجاوب يعمل بكفاءة على جميع أحجام الشاشات
  • سرعة في التحميل وأداء عالي لضمان رضا المستخدمين
  • تصميم يتوافق مع معايير Android Material Design و iOS Human Interface
